<H1>
   esempio FLASK macchina virtuale
</H1>

<H4>
  PRE REQUISITI:
</H4>

<ul>
  <li>python</li>
  <li>venv</li>
  <li>docker</li>
  <li>docker-compose</li>
</ul>

<H4>
  COMANDI DA FARE PER COMINCIARE:
</H4>

<ul>
  <li>python -m venv .venv</li>
  <li>source .venv/bin/activate</li>
  <li>pip install -r requirements.txt</li>
  <li>docker-compose up</li>
  <li>flask run</li>
</ul>
